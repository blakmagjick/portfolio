Group Project - Down To Game

Group project for SEI. Full CRUD app with REST APIs. It's an app to help users find other people to play board games and video games with.
My contributions to the project include: initial set up of the backend server, most of the app stylizing (except for the Profile page), a lot of the code testing and bug fixes, implemented React Router to the frontend, was in charge of monitoring GitHub/dealing with merge conflicts and final deployment.
Built using: Express/React/Mongo
